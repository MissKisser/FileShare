# UI 重构设计文档

## 概述

将现有深色赛博朋克风格升级为简约现代的轻盈色块设计，提升可读性和使用体验。

## 设计风格

| 属性 | 值 |
|------|-----|
| 背景色 | #F8F9FA（淡灰白） |
| 主文字 | #1A1A2E（深灰蓝） |
| 次要文字 | #6B7280（中性灰） |
| 边框色 | #E5E7EB（浅灰） |
| 圆角 | 12px |
| 阴影 | 0 4px 20px rgba(0,0,0,0.06) |
| 间距 | 20px |

## 功能区块配色

| 区块 | 背景色 | 强调色 |
|------|--------|--------|
| 侧边栏 | #FFFFFF | #3B82F6（蓝） |
| 文件上传 | #E8F4FD | #2563EB |
| 文本共享 | #E8F5E9 | #16A34A |
| 存储概览 | #FFFFFF | #F59E0B |
| 上行日志 | #FFF3E0 | #EA580C |

```
+------------------+----------------------------------------+

|  - 新建文本      |  +------------+  +------------+         |

|                  |  +--------------------------------+      |
|                  |  | 存储概览列表                   |      |
|                  |  +--------------------------------+      |
|                  |                                        |
|                  |  +--------------------------------+      |
|                  |  | 上行日志审计                   |      |
|                  |  +--------------------------------+      |
+------------------+----------------------------------------+
```

## 侧边栏设计

- 宽度：200px，固定左侧
- 背景：纯白，左侧一条 4px 蓝色竖条装饰
- Logo 区：标题 "文件暂存" + 版本号
- 快捷按钮：图标 + 文字，hover 时背景变为淡蓝色

## 响应式策略

| 断点 | 布局变化 |
|------|----------|
| >= 1024px | 侧边栏固定 + 双列卡片 |
| < 1024px | 侧边栏隐藏，汉堡菜单触发抽屉 |
| < 640px | 单列全堆叠，统计条改为图标计数 |

## 字体规范

- 标题：Noto Sans SC 700，16px
- 正文：Noto Sans SC 400，14px
- 代码/日志：JetBrains Mono，12px
- 统计数字：Noto Sans SC 700
